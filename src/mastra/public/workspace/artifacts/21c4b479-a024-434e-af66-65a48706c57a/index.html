Mastra Offensive Depth Chart

Think of Mastras primitives as the star players of an AI-powered offense. Each role turns raw model power into an orchestrated scoring drive.

Quarterback

Agents

The field generals. Agents read the defense (your inputs), call audibles with routing logic, and deliver the ball to whatever capability scores fastest.

  • Reads context, goals, and guardrails in real time.
  • Hands off to tools, workflows, or other agents as needed.
  • Streams decisions so humans can override the play.
Drives the huddle
Offensive Line

Workspaces

The protection unit. Workspaces give agents a controlled environment with storage, files, and execution sandboxes, letting plays develop safely.

  • Provides isolated sandboxes per task or customer.
  • Tracks artifacts, logs, and approvals in one place.
  • Absorbs heavy lifting so the QB can stay focused.
Keeps pockets clean
Playbook & Audible Memory

Memory

The accumulated film study. Mastra memories capture prior drives, outcomes, and user preferences so every possession feels tailored.

  • Short-term recall for the current series.
  • Long-term storage for user profiles and outcomes.
  • Observational notes that improve future play-calling.
Knows every blitz
Skill Players

Tools

The receivers and backs who actually rack up yardage. Tools run specialized routes  from API calls to code execution  whenever the agent delivers the ball.

  • Each tool is a discrete capability with clear inputs/outputs.
  • Configured with approval gates for risky plays.
  • Chain together for explosive trick plays (multi-step tasks).
Moves the chains
Coordinated Drives

Workflows

The offensive coordinator's script. Workflows define multi-play sequences  research, draft, edit, send  so the team executes repeatable drives automatically.

  • Breaks big goals into callable phases.
  • Hands off between agents with explicit contracts.
  • Evaluates progress, then calls the next best play.
Controls tempo
Two-Minute Drill

Tasks & Evaluations

Situational packages that keep everyone honest. Tasks pin down a discrete objective, while evaluators grade the result before the clock expires.

  • Atomic assignments that any agent or workflow can pick up.
  • Evaluation hooks flag when to rerun or escalate.
  • Keeps throughput high without sacrificing accuracy.
Finishes drives
