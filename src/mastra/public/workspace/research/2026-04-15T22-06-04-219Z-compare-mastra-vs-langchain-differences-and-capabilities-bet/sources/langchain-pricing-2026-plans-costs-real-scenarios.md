## Summary

Summary tailored to your query: LangChain pricing and licensing, including GitHub/enterprise considerations and paid plans for 2026.

Key points
- Licensing and core: LangChain is open-source (MIT) for its framework and tools (LangChain, LangGraph). The core licenses are free regardless of plan.
- Subscription and pricing model: LangChain uses a usage-based billing layer on top of a SaaS subscription (LangSmith for observability). You pay per traced user request (trace = one request processed), plus monthly subscription for the plan you choose.
- Free tier and plans:
  - Developer (free): 5,000 traces/month included. 1 agent builder limit (50 agent runs/month). Community support. Ideal for solo developers and prototyping.
  - Plus ($39 per user per month): 10,000 traces included. $0.50 per 1,000 overage traces. Email support. Suitable for small-to-mid teams with production apps.
  - Enterprise (custom): Custom pricing and features. Dedicated Customer Success Manager, SLA, and compliance options. Best for large organizations with scale and governance needs.
- Overages: $0.50 per 1,000 traces beyond included monthly allowances. Overage costs can accumulate quickly with high-volume production workloads.
- What’s included across tiers:
  - LangChain framework (MIT license, always free)
  - LangGraph (agents/workflows)
  - LangSmith (observability, debugging, evaluation)
  - Model/provider compatibility: Works with OpenAI, Anthropic, Google, and other providers
- External costs: LLM API usage (OpenAI, Anthropic, Google, etc.) are billed separately by the model provider; these are not included in LangChain subscriptions.
- Practical budgeting guidance:
  - For a 5-person team with moderate usage (10k–50k traces/month): LangSmith costs may run roughly $137–$300/month (including seats and overages).
  - For enterprise-scale deployments processing millions of traces: costs can range from approximately $650–$6,000+/month, depending on scale and optimization.
  - Overall operating costs typically include: LLM API costs (50–70% of budget), cloud infrastructure (15–25%), vector database costs (5–15%), with LangChain subscription being a portion of the total.

Direct answer to your query
- GitHub license: LangChain itself is open-source with MIT license for the core framework; see project repositories for source code and license details.
- Enterprise/commercial pricing: Available as custom, with negotiated pricing, dedicated CSM, and SLAs. Plus plan at $39/user/month; Developer plan free with limits; overages at $0.50/1k traces.
