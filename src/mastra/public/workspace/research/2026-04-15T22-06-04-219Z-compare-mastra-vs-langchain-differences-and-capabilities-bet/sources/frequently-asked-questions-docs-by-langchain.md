- This LangChain docs page covers LangSmith pricing, plans, and billing concepts, plus FAQs that help determine which plan fits your needs.
- Key pricing/plan highlights:
  - Plans available: Developer (individuals), Plus (teams for collaboration), Startup (discounted for early-stage AI apps with generous free traces), Enterprise (advanced admin features, deployment options, annual invoicing). Enterprise requires contacting sales.
  - LangSmith billing starts July 2024 for those already using LangSmith prior to pricing, with options to add seats or more traces by adding a credit card or contacting sales.
  - A “seat” equals a distinct user in your organization; billing is based on total seats.
  - A “trace” is one complete invocation of your application chain/agent, evaluator run, or playground run.
  - A “Deployment Run” covers end-to-end execution of a LangGraph agent deployed via LangSmith Deployment; calls to other LangGraph agents are billed separately to the host deployment, with interrupts treated as separate runs.
  - Individual, per-run pricing for Deployment Runs is $0.005 each; for high-volume usage, sales consultation is recommended for custom options.
  - Ingested events include all trace-related data (inputs/outputs/metadata at start and end of run steps, and feedback).
  - Personal organization starts with 5000 monthly traces; exceeding that requires upgrading plans and adding a credit card.
- Practical steps:
  - Review which plan matches your team size and usage: Developer for individuals, Plus for teams, Startup for startups with favorable terms, Enterprise for advanced needs.
  - If you anticipate high trace volumes or deployment complexity, contact LangChain Sales for enterprise/custom pricing.
  - For continued LangSmith usage beyond free traces, prepare to add payment details as you upgrade features or seats.
