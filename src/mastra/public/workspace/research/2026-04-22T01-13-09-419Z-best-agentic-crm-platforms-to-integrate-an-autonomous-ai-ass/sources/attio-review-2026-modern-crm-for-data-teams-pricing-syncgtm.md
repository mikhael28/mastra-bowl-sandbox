## Summary

Summary tailored to your query:
- Attio is a modern, API-first CRM designed around custom objects and relationship intelligence. It emphasizes flexible data modeling over traditional CRM structures.
- API capabilities
  - Full REST API with real-time webhooks, intended for developer use; strong emphasis on programmatic access and integration.
  - Documentation is developer-friendly, but free plan imposes API rate limits.
  - Notion-like UI and real-time data sync support custom object definitions and relationships.
- Webhook examples and automation
  - Webhooks are available for real-time updates, enabling event-driven workflows; practical use depends on plan and rate limits.
  - Automation/workflow capabilities are more mature on higher-tier plans; AI features for automation are limited on core plans.
- Custom objects and data modeling
  - Core strength: define any object type, with custom fields and inter-object relationships.
  - Free plan restricts to up to 3 custom objects; impact on larger data models requires paid plans.
- Email/calendar integration and relationship intelligence
  - Relationship intelligence automatically captures email/calendar interactions when connected, enabling richer relationship graphs.
  - Full value relies on having active email/calendar connections; setup friction exists.
- AI features and attributes
  - AI-generated attributes for classification, summarization, and research are available, primarily on Pro plans and above.
- Automation/workflow limits
  - Attio supports automation, but some advanced automation capabilities and limits depend on plan tier; users may encounter limitations compared to traditional CRM automation tools.
- Email sync behavior
  - Email sync is central to relationship intelligence; effectiveness improves with connected accounts but may have privacy and security considerations for some teams.
- Integrations and ecosystem
  - Notions-like, but native integrations are relatively limited; many tools require Zapier or custom API work to bridge gaps.
- Pricing and tiers (high level)
  - Free: up to 3 users with up to 50,000 records.
  - Paid: Plus at $29/user/month; Pro at $59/user/month (annual billing). Higher tiers unlock more AI features, automation, and API usage.
- Best-fit use cases
  - Data-driven startups and growth-stage teams needing a flexible, API-first CRM with custom objects, robust developer tooling, and a generous free tier.
  - Teams ready to accept a learning curve and some missing native integrations, in exchange for modeling flexibility and a modern UI.
