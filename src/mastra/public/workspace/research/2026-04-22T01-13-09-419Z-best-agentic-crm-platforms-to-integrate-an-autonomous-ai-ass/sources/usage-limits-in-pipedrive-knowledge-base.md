## Summary

Here’s a concise, user-focused summary of the Pipedrive usage limits page, tailored to your query about API rate limits, custom objects, workflow/action limits, email sync behavior, and pricing tiers, including vendor docs and independent reviews.

What the page covers (highlights)
- Plan-based usage limits: Pipedrive usage caps depend on whether you’re on old or new plans. There are detailed per-feature limits tied to seat counts, user limits, and per-company quotas, plus behavior changes reflected as of late 2025.
- Core limits by plan:
  - Old plans (Essential, Advanced, Professional, Power) and new plans (Lite, Growth, Premium, Ultimate) each have defined ceilings for:
    - Users/seats, emails, and types of usage (e.g., leads/deals, custom fields, insights, automations, sequences, email sync, shared inboxes, teams, visibility/permissions, data enrichment, scores, API tokens).
    - Some features are per company (e.g., leads/deals, automations, sequences, data enrichment) while others are per user or per plan tier.
  - Some limits show high ceilings or “N/A” where not applicable, and “up to 100 million” seat-usage caps appear in the new plan tables for scalable deployments.
- Featured items with limits (per company or per user):
  - Leads and deals, custom fields, insights reports, automations, if/else conditions, sequences, multiple email sync, shared team inboxes, teams, visibility groups, permission sets, data enrichment, custom scores, API tokens.
- Per-plan examples:
  - New plans (Lite, Growth, Premium, Ultimate) show seat allowances (ranging from 2,500 up to 30,000+) and associated limits for per-company features (e.g., leads/deals, custom fields, insights, automations).
  - Email-related limits vary by plan (number of email accounts, per-user vs per-company allowances).
- API and automation notes:
  - API tokens are listed as a per-company limit item under Enterprise or per plan, indicating a cap on API usage or credential scope depending on the plan.
  - Automations and sequences have per-automation or per-company caps, and If/Else conditions in workflows are also limited by plan.
