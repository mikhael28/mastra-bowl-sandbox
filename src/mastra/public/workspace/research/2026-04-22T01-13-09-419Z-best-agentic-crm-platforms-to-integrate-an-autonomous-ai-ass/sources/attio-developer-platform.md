## Summary

Summary:

The Attio Developer Platform provides a comprehensive suite to extend, automate, and integrate Attio via three main cores:

- REST API and OAuth: Secure, scalable REST API with full read/write access, OAuth 2.0, webhooks, and extensive documentation for bi-directional data sync and CRM control.

- App SDK (TypeScript & React): Build custom apps inside Attio with ready-made UI components, forms, server functions, authentication, webhooks, and UI widgets. Supports multiple entry points (record actions, widgets, text selection) and publishing to internal workspaces or the Attio App Store. Emphasizes a seamless bridge between backend logic and frontend UI.

- MCP Server: Aimed at AI agents and automation tools, enabling agent-driven workflows with read/write access, structured tool calls, and multi-tool orchestration. Features secure OAuth, human-readable responses, one-click installation for Claude/Notion, and AI-centric data access.

Key capabilities aligned to your query:
- Webhooks and event-driven architecture for automation and real-time sync.
- API endpoints for full data access, including custom objects and schemas via the App SDK and REST API.
- Custom object support through the App SDK and REST, enabling bespoke data models within Attio.
- Automation primitives via MCP servers and server functions to drive agent-based workflows.
- Email-sync implementation readiness via API/event plumbing and UI integration (via App SDK and webhooks).
- Comprehensive developer docs and sample apps (including open-source Linear app) to accelerate integration.

If you’re evaluating for specific needs:
- For automation and AI agent workflows: MCP server capabilities with structured tool calls and OAuth-secured access.
- For custom apps and UI integrations: App SDK with TypeScript/React, forms, server functions, and pre-built UI components.
- For data integration and external systems: REST API with full CRUD, webhooks, and bi-directional sync.
- For object customization: Custom object support via REST and SDK, with publishing options to App Store.
