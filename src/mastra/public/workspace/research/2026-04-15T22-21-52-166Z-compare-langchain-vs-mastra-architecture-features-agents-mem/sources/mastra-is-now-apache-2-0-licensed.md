## Summary

Mastra has switched to an open-source Apache 2.0 license. Key points relevant to your query:

- Current license: Apache License 2.0 (permissive open-source license).
- Prior license: ELv2 (used earlier in January), but the project explicitly moved to Apache 2.0.
- What Apache 2.0 allows: use in commercial products and services, modify and distribute original or modified code, use privately without restrictions; must include the original Apache 2.0 license text.
- Implications for users/organizations: easier to adopt, modify, and contribute; reduces friction for enterprise use while preserving attribution.
- How it differs from prior license: Apache 2.0 is more permissive for many enterprise use cases than ELv2.
