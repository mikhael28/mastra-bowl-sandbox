## Summary

Summary:

This GitHub repository, aymeric-roucher/agent_reasoning_benchmark, provides a reusable benchmarking engine to compare AI agent systems across frameworks and benchmarks. It supports testing with LangChain and Transformers agents, and includes:

- Benchmarks: GAIA, a custom agent-reasoning benchmark (with tasks from GSM8K, HotpotQA, GAIA), and other curated evaluation tasks.
- Models and frameworks: evaluate agents on multiple frameworks (LangChain, Transformers agents) and models.
- Reproducibility: an emphasis on reproducible benchmarks with test harnesses, data, and evaluation flows.
- Evaluation: supports LLM-judge evaluation and parallel processing to speed up results.
- Artifacts: codebase for CI-ready workflows, raw data, and instructions to reproduce results.
- Language and licensing: primarily Python, with Jupyter Notebooks; Apache-2.0 license.

If you’re specifically looking for reproducible benchmark repositories or scripts to compare LLM agent frameworks (LangChain vs Mastra) with raw data, test harnesses, CI artifacts, and setup instructions for 2026, this repository serves as a centralized engine and reference implementation, including cross-framework evaluation and parallelized judging.
