## Summary

Summary:

- This page explains how to connect Accredible certificates to Mastra AI using Composio’s MCP (Model Context Protocol) framework.
- Key outcome: build an Accredible certificates agent within Mastra AI that can bulk-create certificates, generate PDFs, list templates, clone groups for new cohorts, and manage credentials via a toolRouter session.
- Core concepts:
  - Mastra AI: TypeScript framework for building AI agents with tool support and MCP client integration.
  - Accredible MCP server: Enables secure, structured access to Accredible actions (issuing, grouping, PDF generation, templates).
- Practical capabilities enabled by the integration:
  - View and select certificate templates.
  - Generate and export PDFs (single or bulk).
  - Manage evidence items and references for certificates.
  - Automated group/collection management (create, clone, delete).
  - Bulk credential creation and issuance.
- Supported tools within the MCP integration (examples):
  - Bulk Create Credentials (V2)
  - Clone Group
  - Create/Delete/Update Group
  - Create/Delete/Edit Reference
  - Generate PDFs for Credentials
  - List Templates
  - Search Collections

User-specific note: The page is a setup/tutorial for linking Accredible with Mastra AI via Composio, enabling automated management of certificates through AI agents. If you’re specifically seeking a 2026 SOC 2 Type II PDF report on Mastra, this page does not cover that content.
