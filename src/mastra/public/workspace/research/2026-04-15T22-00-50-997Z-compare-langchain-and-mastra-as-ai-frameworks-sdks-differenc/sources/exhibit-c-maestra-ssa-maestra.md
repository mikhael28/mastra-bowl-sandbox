## Summary

Summary:

- Exhibit C outlines Maestra’s information security posture for SOC 2 Type II compliance, with emphasis on data protection, risk management, access controls, data handling, retention, and disposal.
- Key controls described:
  - Formal information security policy aligned to regulatory/business requirements.
  - Annual (or change-driven) enterprise risk assessment and threat/vulnerability management.
  - Confidentiality agreements for employees and strict access controls for privileged users.
  - Data Handling, Retention, and Disposal Program with data classification and retention schedules.
  - Data minimization and legal/ contractual obligation considerations for data retention.
  - Network segmentation to shield confidential data, especially in cloud environments.
  - Vulnerability management, including internal/external scans, remediation, and continuous monitoring.
  - Independent third-party penetration testing at least annually for systems with confidential data or high risk.
- What this means for your needs:
  - SOC 2 Type II alignment is described, but no explicit HIPAA or GDPR attestations are stated within Exhibit C itself.
  - No specific data residency commitments or official statements on where data is stored or processed are included in this excerpt.
  - Security controls (policy, risk assessment, access controls, data handling, segmentation, and ongoing testing) are described as part of the SOC 2 framework and security program.
